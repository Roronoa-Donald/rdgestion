# AGENTS.md

Ce fichier explique l'organisation des documents Markdown du projet RDGESTION.

## Dossier `md/`

Tous les fichiers Markdown (`.md`) du projet sont centralisés dans le dossier `md/` à la racine, à l'exception de :
- `RDGESTION/` qui contient la documentation originale du cahier des charges (00-24)
- Les fichiers `README.md` éventuels

Cette organisation permet de :
- Retrouver facilement tous les documents de suivi
- Nettoyer rapidement les fichiers temporaires
- Éviter de polluer la racine du projet

## Fichiers dans `md/`

- `CLAUDE.md` — Guide pour Claude Code
- `conv.md` — Notes de conversation
- `planing.md` — Planification du projet
- `PRODUCT.md` — Description du produit
- `test_gui.md` — Rapport de tests GUI
- `avance.md` — Rapport d'avancement vs cahier des charges
